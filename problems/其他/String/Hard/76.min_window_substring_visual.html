📘 LeetCode 76 - Minimum Window Substring

🔍 題目說明

給定字串 s 和 t，找出 s 中最短子字串使得其中包含 t 中所有字母（順序不限、可重複）。

📘 範例

Input:  s = "ADOBECODEBANC", t = "ABC"
Output: "BANC"

給定字串 s 和 t，找出 s 中最短子字串使得其中包含 t 中所有字母（順序不限、可重複）。

🧠 解法思路

  • 使用滑動窗口 (sliding window)
  • 右指針擴展字串直到包含所有目標字母
  • 左指針縮小字串以求最短答案

⏱️ 時間與空間複雜度

  • 時間複雜度：O(m + n)
  • 空間複雜度：O(n)

💡 Python 程式碼

import collections
class Solution:
    def minWindow(self, s, t):
        freq = dict(collections.Counter(t))
        count = 0
        slow = 0
        min_str = ""
        min_len = float("inf")
        for fast in range(len(s)):
            c = s[fast]
            if c in freq:
                freq[c] -= 1
                if freq[c] == 0:
                    count += 1
            while all(v <= 0 for v in freq.values()):
                if (fast - slow + 1) < min_len:
                    min_len = fast - slow + 1
                    min_str = s[slow:fast+1]
                left_c = s[slow]
                slow += 1
                if left_c in freq:
                    freq[left_c] += 1
                    if freq[left_c] > 0:
                        count -= 1
        return min_str
