🚶‍♂️ 題目敘述：70. 爬樓梯 (Climbing Stairs)

你正在爬一座樓梯。需要 n 步你才能到達樓頂。

每次你可以爬 1 步或 2 步。你有多少種不同的方法可以爬到樓頂？

🧪 範例

  • 輸入： n = 2
    輸出： 2
    解釋：1+1 或 2
  • 輸入： n = 3
    輸出： 3
    解釋：1+1+1、1+2、2+1

📌 限制條件：

  • 1 <= n <= 45

🧠 解題思路：費波那契數列

當你走到第 n 階時，其實只有兩種情況：

  • 你是從第 n - 1 階走一步上來的
  • 你是從第 n - 2 階走兩步上來的

也就是說：

f(n) = f(n - 1) + f(n - 2)

初始條件：

f(1) = 1
f(2) = 2

Climbing Stairs DP Visualization

1: if n <= 2: return n
2: a, b = 1, 2
3: for _ in range(3, n + 1):
4:     a, b = b, a + b
5: return b
