🧭 LeetCode 62: Unique Paths

在 m x n 的網格中，從左上走到右下，每次只能向右或向下，有多少種走法？
dp[i][j] = dp[i-1][j] + dp[i][j-1]
🧠 時間複雜度：O(mn)
🧠 空間複雜度：O(mn)

🔧 原始程式碼


class Solution:
    def uniquePaths(self, m: int, n: int) -> int:
        dp = [[1]*n for _ in range(m)]
        for i in range(1, m):
            for j in range(1, n):
                dp[i][j] = dp[i-1][j] + dp[i][j-1]
        return dp[m-1][n-1]
