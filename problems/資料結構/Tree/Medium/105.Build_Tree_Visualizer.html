class Solution:
    def buildTree(self, preorder: List[int], inorder: List[int]) -> Optional[TreeNode]:
        if not preorder or not inorder:  # 空的代表沒節點
            return None

        root = TreeNode(preorder[0])
        divide_index = inorder.index(preorder[0])

        # 左子樹：前序從 1 開始取 divide_index 個；中序取左邊
        root.left = self.buildTree(preorder[1: divide_index + 1], inorder[:divide_index])

        # 右子樹：剩下部分
        root.right = self.buildTree(preorder[divide_index + 1:], inorder[divide_index + 1:])

        return root

📘 解法說明

  • 前序遍歷的第一個元素為根節點
  • 在中序遍歷中找到根節點位置，左邊就是左子樹，右邊是右子樹
  • 根據子樹長度從前序中切出對應子陣列，遞迴重建左右子樹

💡 Tips

  • 前序順序：root → left → right
  • 中序順序：left → root → right
  • 可使用 hash map 優化 index 查找時間（中序值對位置）

⏱ 時間與空間複雜度

  • 時間複雜度：O(n)
  • 空間複雜度：O(n)

每個節點處理一次，並儲存在遞迴與新建節點結構中。
