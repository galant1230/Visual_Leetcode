🌟 230. BST 中找第 K 小元素 - 中序動畫


尚未開始

🔧 Python 解法

class Solution:
    def kthSmallest(self, root: Optional[TreeNode], k: int) -> int:
        inorder_result = []

        def InOrder(root):  # 左中右：中序遍歷
            if not root:
                return
            InOrder(root.left)
            inorder_result.append(root.val)
            InOrder(root.right)

        InOrder(root)
        return inorder_result[k - 1]

📘 解法說明

  • 中序遍歷 BST → 結果是 遞增排序
  • 將結果存進陣列後，直接回傳第 k-1 個即可

💡 Tips

  • 若要多次查詢第 k 小，應使用「augmented tree + subtree size」優化
  • 本方法簡潔直觀，適合單次查詢
  • 如要節省記憶體，可在找到第 k 個值後立刻返回（加計數器）

⏱ 時間與空間複雜度

  • 時間複雜度：O(n)
  • 空間複雜度：O(n)

需完整中序遍歷一次（n 節點），且陣列儲存所有節點值。
